# Cloud-API-Security-Monitor
A Dockerised cloud security system for detecting and logging suspicious API requests.

## The Goal Of This Project 

Receives API request data, Then Scan it for suspicious patterns, Assigns a risk score, Returns a security decision, Logs the event.

Skills Used

* Python
* FastAPI
* Docker
* API security
* logging
* DevSecOps
* GitHub Actions
* cloud deployment


Goal of the API is to dectect - Prompt Injection, Secreat Leakage, Dangerous Commands, SQL Injection Patterns, XSS patterns, Oversized Inputs, Suspicious Encoding.
